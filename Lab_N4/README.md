## Задача

Реализовать класс `LineSegment` c полями: `double x`, `double y`, которые описывают координаты начала и конца отрезка на координатной прямой. Реализовать метод, который определяет попадает ли заданное число в отрезок, результат должен быть типа `bool`.<br>
Перегрузить: 
<ul><i>Унарные операторы</i></ul>

`!` -- вычеслить длину торезка, результат должен быть типа `double`

`++` -- увеличить координаты границ отрезка на 1
    
<ul><i>Операции приведения типа</i></ul>

`int` (яваная) -- результатом является целая часть координаты x  

`double` (неявная) -- рузультатом является координата y
    
<ul><i>Бинарные операции</i></ul>  

`+ int d` -- координаты границ увеличиваются на число d<br>
`> int d` -- результат равен true, если целое число попадает в заданный отрезок и false-в противном случае
