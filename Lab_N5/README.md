### Задание 1. Задача 5
`Из исходного файла получить новый файл, исключив повторные вхождения чисел.`

#### Пример
Текстовый файл:
    2,2,4,5,1
Выходной файл:
    2,4,5,1

#### Алгоритм решения
Считываем цифры из файла. Заполняем словарь где `ключ` - цифра, `значение` - сколько раз встретилась. Если значение = 1, то записываем в выходной файл.

### Задание 2. Задача 5
`Скопировать элементы заданного файла в квадратную матрицу размером n×n (если элементов
файла недостает, заполнить оставшиеся элементы матрицы нулями). Заменить все строки
таблицы на строку с максимальным произведением элементов.`

#### Пример
Исходная матрица:   
2  2
1  1

Матрица результата:
2   2
2   2

#### Алгоритм решения
Ищем строку с максимальным произведением и меняем все другие строки на эту строку

### Задание 3. Задача 5
`Файл содержит сведения об игрушках: название игрушки, ее стоимость в рублях и возрастные
границы (например, игрушка может предназначаться для детей от двух до пяти лет). Получить
название игрушек, цена которых не превышает k руб. и которые подходят детям 5 лет`

#### Пример
Игрушка 1: `Car, 30, 3 ,7`
Игрушка 2: `Sword, 50, 5, 10`
Игрушка 3: `Train, 40, 4, 8`
k = 45

Результа:
Car, Train

#### Алгоритм решения
Считваем структуру из файла. Проверяем если цена игрушку <= k && minAge <= 5 && maxAge >= 5. Если верно выводи название игрушки

### Здание 4. Задача 5
`Для заданного файла возвратить true, если он содержит заданное число b, и false в противном
случае.`

#### Пример
Файл: 4,2,4,3,6,7,1
b = 1

Результат: Содержит

#### Алгоритм решения
Считываем числа из файла и если считали искомое число `b`, то выводим содрежит. Если так и не считали число, то выводим не содержит

### Задание 5. Задача 5
`Вычислить сумму элементов, которые кратны заданному числу k`

#### Пример
Файл: 4,2,4,3,6,7,1
k = 2

Результат:  16

#### Алгоритм решения
Считываем числа из файла и если считали число кратное `k`, то суммируем. Выводим сумму

### Задание 6. Задача 5
`Переписать в другой файл строки, в которых нет цифр`

#### Пример
Файл:
privet
pochemy
3 42 1 vot takaya
koala!021

Результат:
privet
pochemy

#### Алгоритм решения
Построчно считвыем файл. В каждой строке считваем каждый символ. Если символ является цифрой, то записываем эту строку в выходной файл
