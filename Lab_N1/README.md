## Задачи

### Задание 1. Задача 5

`Двузначное.
Дана сигнатура функции: bool is2Digits (int x);
Необходимо реализовать функцию таким образом, чтобы она принимала число
x и возвращала true, если оно двузначное
`
##### Пример 1:
x=32
результат: true
##### Пример 2:
x=516
результат: false

#### Описание решения

Делим нацело искомне число на 10. Если получили число ∈ [1;9] , то искомое число двузначное

### Задание 2. Задача 5

`Тройной максимум.
Дана сигнатура функции: int max3 (int x, int y, int z);
Необходимо реализовать функцию таким образом, чтобы она возвращала
максимальное из трех полученных функцией чисел. Подсказка: идеальное
решение включает всего две инструкции if и не содержит вложенных if.
`
##### Пример 1:
x=5 y=7 z=7
результат: 7
##### Пример 2:
x=8 y=-1 z=4
результат: 8

#### Описание решения

Сравниаем сначала второе чилсо с первым, если оно больше то меняем значеие первого числа на второе.
Дальше сравниваем третье число с первым, если оно больше то меняем значение первого числа на третье

### Задание 3. Задача 5

`Длина числа.
Дана сигнатура функции: int numLen (long x);
Необходимо реализовать функцию таким образом, чтобы она возвращала
количество знаков в числе x.
`
##### Пример 1:
x=12567
результат: 5
##### Пример 2:
x = 0
результат: 1

#### Описание решения
Создаем перменную счетчик типа `int` присваем ей значение 0.
В цикле `do while` прибавояем переменную счетчик на 1 и делим искомое число на 10.
Таким образом посчитается длина числа.

### Задание 4. Задача 5

`Добавление массива в массив.
Дана сигнатура функции: int * add (int arr[], int ins[], int pos);
Необходимо реализовать функцию таким образом, чтобы она возвращала новый
массив, который будет содержать все элементы массива arr, однако в позицию pos
будут вставлены значения массива ins
`

#### Описание решения
Используем `vector` - динамичский массив
Инициализураем 2 произвольных массива. Получаем индекс вставки
Инициализурем 3 массив.
Цикл `for` по длине первого массива. Если `i != позиции вставки` добавляем в 3 массив элементы 1 массива
В противном случае запускаем цикл `for` по длине второго массива и добавляем его элементы в 3 массив.

##### Пример:
arr=[1,2,3,4,5]
ins=[7,8,9]
pos=3
результат: [1,2,3,7,8,9,4,5]
